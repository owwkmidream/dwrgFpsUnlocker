# 第五人格帧率解锁
解锁并设定第五人格(PC互通版)帧率上限

### 特性
1. 跟随游戏启动，启动时自动应用上次帧数
2. 长时间未操作自动关闭进程访问（最小侵入）
3. 无需拨动滑块[^1]，甚至能在登入账号前设定

### 说明

![ui](resource/ui.png)

在下拉框填写正整数fps值，或选择一个预设，回车↵或单击“应用”按钮设定帧率。<br>
点亮数值标签可查看实时帧率（应用帧率时这里也会临时显示以方便确认）<br>

勾选单选圆钮程序会在关闭时记忆fps、并在下次启动时自动应用（注意：部分版本会创建名为hipp的文件以保存）

按快捷键`F5`与默认帧率切换

设好后关闭程序没有影响（不过保留托盘可以跟随启动）.

> *注意：游戏中<strong>局内外高帧率滑块状态需一致</strong>，否则帧率会在进出对战时被重设*
